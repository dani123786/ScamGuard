# Scam Awareness Videos

## 📁 Folder Structure

Place your scam awareness videos in this folder with the following naming convention:

```
static/videos/
├── phishing.mp4
├── cryptocurrency.mp4
├── investment.mp4
├── tech_support.mp4
├── online_shopping.mp4
├── identity_theft.mp4
├── lottery.mp4
├── employment.mp4
├── social_media.mp4
└── deepfake.mp4
```

## 📹 Video Requirements

### File Naming
- Use the exact scam type key from the application
- Use lowercase with underscores
- Supported formats: MP4, WebM, OGG

### Video Specifications
- **Format:** MP4 (H.264) recommended for best compatibility
- **Resolution:** 1280x720 (720p) or 1920x1080 (1080p)
- **Duration:** 2-5 minutes recommended
- **File Size:** Keep under 50MB for faster loading
- **Aspect Ratio:** 16:9 recommended

### Content Guidelines
- Clear explanation of the scam type
- Real-world examples
- Warning signs to watch for
- Prevention tips
- Professional narration
- Subtitles recommended for accessibility

## 📝 How to Add Videos

1. **Prepare your videos** according to the specifications above
2. **Rename them** using the exact names from the table
3. **Copy them** to this folder (`static/videos/`)
4. **Restart the application** if it's running
5. Videos will automatically appear on scam detail pages

## 🎥 Video Sources

You can create or obtain videos from:
- Educational content creators
- Government awareness campaigns
- Cybersecurity organizations
- Your own recordings
- Licensed stock footage

## ⚠️ Important Notes

- Ensure you have rights to use the videos
- Videos should be appropriate for all audiences
- Keep file sizes reasonable for web delivery
- Test videos in different browsers
- Consider adding captions for accessibility

## 🔧 Troubleshooting

**Video not showing?**
- Check file name matches exactly (case-sensitive on some systems)
- Verify file format is supported (MP4 recommended)
- Check file permissions
- Clear browser cache

**Video not playing?**
- Ensure codec is H.264 for MP4
- Try converting with a tool like HandBrake
- Check browser console for errors
